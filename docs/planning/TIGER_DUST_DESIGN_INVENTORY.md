# 4. Missing or Weak Assets

These are the gaps most likely to reduce site quality.

## 4.1 People imagery
Need:
- founder/staff portrait(s)
- human presence in the space
- optional process/studio shots

Why:
The brand currently risks feeling like pure object collage without enough human authorship.

## 4.2 Clean storefront coverage
Need:
- daylight exterior
- dusk/night exterior if strong
- tighter window/signage crops
- entrance approach shot

Why:
This is the most functional public-facing image set for a brick-and-mortar shop.

## 4.3 Signature interior wide
Need:
- one or two definitive room-wide shots

Why:
The “wonder room” experience is central to the brand and should be captured intentionally.

## 4.4 Jewelry / handmade work coverage
Need:
- small-object closeups
- display/staging images
- process-related supporting images if useful

Why:
This is part of the business identity but may not read strongly enough in current public imagery.

## 4.5 Pinball / arcade / playful layer
Need:
- clear image(s) representing this aspect of the shop

Why:
It is part of the copy and personality, but can get visually drowned out by taxidermy/oddities.

# 5. Migration Guidance for Website v2

## Keep
- usefulness of the current homepage
- obvious hours/address/contact/maps access
- dark branded tone
- straightforward navigation

## Improve
- image quality/consistency
- public visual hierarchy
- freshness of gallery-like content
- stronger human presence
- intentional category imagery

## Avoid
- overcomplicated public browsing
- generic ecommerce visuals
- sanitizing the weirdness out of the brand
- replacing useful practical information with pure mood

# 7. Summary

Tiger Dust already appears to have enough raw visual material to build a strong public-facing site, but the materials need to be organized into a coherent system.

The critical distinction is:
- **Tiger Dust folder** = canonical brand identity
- **House of Epicurus folder** = supporting archive and aesthetic reference

The next step is not more abstract design discussion.
The next step is to:
1. normalize the brand assets
2. identify current usable photography
3. fill the missing image gaps with intentional assignment briefs
